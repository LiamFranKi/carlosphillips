# Capacitación Docente — IE 5084 Carlos Phillips

Web de capacitación sobre **NotebookLM** para docentes de primaria.

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm run preview
```

Los recursos (logo, podcast, presentación e infografía) viven en `public/assets/`.

Las respuestas del Pre-Test y Post-Test se envían a Google Sheets mediante Apps Script.
